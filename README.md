# Интернет-магазин эксклюзивных вещей
Данный проект — моя дипломная работа, в рамках которой я создал интернет-магазин по продаже эксклюзивных товаров. Целью проекта является разработка современных и удобных веб-приложений для электронной коммерции с учетом просмотра каталога продуктов, оформления заказов и управления ассортиментом.

В этой репозитории содержится исходный код сайта, включая фронтенд и бэкенд, а также все необходимые материалы для развертывания и дальнейшей доработки проекта.

Проект требует навыков веб-разработки, работы с базами данных, организации пользовательского интерфейса.
